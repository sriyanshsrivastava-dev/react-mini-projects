
# React Mini Projects 

A collection of small React projects built using Vite and Tailwind CSS for practice, experimentation, and improving frontend development skills.

---

## 📌 About

This repository contains multiple standalone mini projects.  
Each project focuses on learning a specific concept, UI pattern, or feature in React.

---

## 🛠️ Tech Stack

- React
- Vite
- Tailwind CSS
- JavaScript (ES6+)

---

## 📂 Project Structure

```

react-mini-projects/
│
├── project1/
├── project2/
├── project3/
│— .gitignore
└── README.md

````

Each folder is an independent React project.

---

## ▶️ How to Run a Project

1. Clone the repository:
```bash
git clone https://github.com/sriyanshsrivastava-dev/react-mini-projects.git
````

2. Navigate to any project:

```bash
cd project1
```

3. Install dependencies:

```bash
npm install
```

4. Start the development server:

```bash
npm run dev
```

5. Open in browser:

```
http://localhost:5173
```
